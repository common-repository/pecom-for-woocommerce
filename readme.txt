Automatically calculate the shipping cost for PECOM tariffs

== Description ==

The official PEC integration application.

The solution allows you to integrate your website with your personal account of the PEC. After installation, the PEC delivery service will be added, the inclusion of which will allow:

Online:
* Choosing a delivery address or pickup point on the map;
* Receive delivery time and cost;

In the plugin admin panel:
* Declared value indication;
* Indication of the presence of protective shipping packaging;


== Installation ==

= From your WordPress dashboard =

Visit 'Plugins > Add New'
Search for 'Pecom for WooCommerce'
Install and activate Pecom for WooCommerce from your Plugins page.

Then create new Shipping Zone and add PECOM as a method.

== Frequently Asked Questions ==

= How accurate is this plugin? =

The plugin by itself has no methods to calculate the shipping price. All data comes from official API PEC
